# SQLite Viewer

A lightweight web-based tool for viewing and exploring SQLite databases.

## Features

- Browse database tables
- View table contents
- Execute SQL queries
- Export data
- User-friendly interface

## Installation

1. Clone the repository
2. Place files in your web root
3. Open in a web browser

## Usage

1. Upload or select a SQLite database file
2. Browse tables and execute queries
3. View results in real-time

## Requirements

- PHP 7.0+
- SQLite3 support
